此外，单个线程中的异常需要自己处理掉。无法被main方法处理，即使该线程run()内抛出RunException()，main()中使用了catch，
也是无法捕获的。

可以这么处理：在创建ExecutorsService的时候，使用ThreadPool这个参数，写一个匿名内部类，指定新建线程的方法，在每个线程建立后，
指定该线程的未捕获异常的处理类。
